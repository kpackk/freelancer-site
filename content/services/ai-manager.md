# ИИ-менеджер для магазина на GPT

Чатбот, который заменяет менеджера по продажам: отвечает на вопросы клиентов 24/7, принимает заявки, консультирует по каталогу и передаёт горячих лидов оператору. Работает в Telegram, WhatsApp и виджете на сайте одновременно.

## Что входит

- GPT-чатбот на базе OpenAI API (GPT-4o / GPT-4 Mini)
- Интеграция с каталогом товаров и FAQ
- Подключение к Telegram, WhatsApp (WhatsApp Business API)
- Виджет для сайта (чат в правом углу)
- Приём и квалификация заявок
- Уведомления менеджеру при горячем лиде
- Подключение к CRM (AmoCRM, Bitrix24, Google Sheets)
- Настройка персонального тона и скриптов
- Аналитика диалогов

## Стек

OpenAI API (GPT-4o) · Python (aiogram / FastAPI) · Telegram Bot API · WhatsApp Business API · n8n · PostgreSQL · Redis
