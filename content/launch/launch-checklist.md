# Чеклист запуска сайта Ruslan WEB

Плейсхолдер для домена: `{SITE_URL}` — заменить после деплоя на Vercel.

---

## За 1–2 дня до запуска

### Технические проверки
- [ ] Сайт задеплоен на Vercel, SITE_URL известен
- [ ] HTTPS работает (SSL-сертификат активен)
- [ ] Все страницы открываются без ошибок 404/500
- [ ] Мобильная версия корректна (проверить на реальном iPhone и Android)
- [ ] Формы обратной связи работают и уведомления приходят в Telegram
- [ ] PageSpeed Insights: Desktop ≥ 80, Mobile ≥ 70
- [ ] Удалить/скрыть файл `content/blog/hello-world.mdx` (плейсхолдер)

### SEO-базис
- [ ] sitemap.xml доступен по адресу `{SITE_URL}/sitemap.xml`
- [ ] robots.txt доступен по адресу `{SITE_URL}/robots.txt`
- [ ] Все страницы имеют уникальные `<title>` и `<meta description>`
- [ ] JSON-LD Schema.org установлены (Person, LocalBusiness, Service)
- [ ] Open Graph теги для всех страниц

### Аналитика
- [ ] Яндекс.Метрика подключена и собирает данные
- [ ] Google Analytics 4 (GA4) подключён
- [ ] Цели в Метрике настроены: клик по Telegram, отправка формы
- [ ] Верификация в Google Search Console (добавить сайт)
- [ ] Верификация в Яндекс.Вебмастер (добавить сайт)

### Контент
- [ ] Все ссылки в портфолио рабочие (espaniero.ru, assoro.ru, exactcarbon.ru, danila-master.ru)
- [ ] Контакты актуальны: Telegram @ruslan_web_dev, email hello@ruslanweb.ru
- [ ] Ссылки на Kwork и FL.ru в About обновлены на реальные URL профилей
