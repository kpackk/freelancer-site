## Что такое VK-бот и как он работает

VK-бот — это программа, которая автоматически отвечает на сообщения в вашем сообществе ВКонтакте. Бот подключается через официальный [VK Bots API](https://dev.vk.com/ru/api/bots/getting-started) и получает сообщения пользователей через механизм Callback API или Long Poll.

Когда пользователь пишет в сообщения сообщества, бот:
1. Получает уведомление о новом сообщении
2. Обрабатывает текст и определяет нужный ответ
3. Отправляет ответ обратно пользователю

Всё это происходит за доли секунды и не требует участия живого человека.

### Что может VK-бот

- Отвечать на типовые вопросы (цены, адрес, режим работы)
- Принимать заявки и собирать контакты
- Показывать каталог товаров и оформлять заказы
- Проводить опросы и квизы
- Рассылать сообщения подписчикам (через VK Notify или рассылки)
- Интегрироваться с CRM, 1С, внешними API
- Работать с кнопками, карточками товаров, вложениями

## VK-бот vs Telegram-бот: главные отличия

Оба инструмента позволяют автоматизировать общение с клиентами, но технически и функционально они существенно отличаются.

### Аудитория

| Параметр | VK | Telegram |
|----------|-----|---------|
| Аудитория в России | ~100 млн пользователей | ~80 млн пользователей |
| Возрастной профиль | Более широкий (16–55+) | Смещён к 18–40, IT/бизнес |
| Типичный бизнес | Розница, услуги, локальный бизнес | B2B, IT, доставка, e-commerce |
| Точка контакта | Сообщества ВКонтакте | Личный канал/бот |

**Ключевой критерий:** если ваша аудитория уже подписана на ваше VK-сообщество — VK-бот позволяет работать прямо там. Если клиенты активны в Telegram — логичнее идти туда.

### Технические возможности

| Возможность | VK Bot API | Telegram Bot API |
|-------------|------------|-----------------|
| Кнопки под сообщением | ✅ Keyboard + Inline | ✅ ReplyKeyboard + InlineKeyboard |
| Карточки с изображением | ✅ Карусель | ✅ Inline-режим / фото |
| Платежи | ✅ VK Pay | ✅ Telegram Payments (ЮКасса, Stripe) |
| Рассылки | ✅ Через рассылки сообщества | ✅ Через broadcast (нет официального) |
| Боты в группах | ❌ Только в личных сообщениях сообщества | ✅ В группах и каналах |
| Мини-приложения | ✅ VK Mini Apps | ✅ Telegram Mini Apps |
| Голосовые сообщения | ✅ | ✅ |
| Webhook | ✅ Callback API | ✅ setWebhook |
| Polling | ✅ Long Poll | ✅ getUpdates |

### Ограничения и нюансы

**VK:**
- Боты работают только в сообществах (не в личных переписках между пользователями)
- Пользователь должен **разрешить** рассылки — нельзя писать тем, кто не подписался
- Rate limit: 20 сообщений в секунду на сообщество
- Требуется верификация сообщества для некоторых функций

**Telegram:**
- Бот может писать только тем, кто первым написал боту (или вступил в канал)
- Нет прямого доступа к аудитории ВКонтакте
- Больше гибкости для сложных сценариев (FSM, inline-клавиатуры, deep linking)

## Технологии разработки VK-бота

### Python + vkbottle / vk_api

Наиболее популярный стек в 2026 году:

```python
import asyncio
from vkbottle.bot import Bot, Message

bot = Bot(token="ваш_токен_сообщества")

@bot.on.message(text="Привет")
async def greet_handler(message: Message):
    await message.answer("Привет! Чем могу помочь? Напиши /услуги или /цены")

asyncio.run(bot.run_polling())
```

**vkbottle** — современная async-библиотека с удобным декораторным синтаксисом. Поддерживает хендлеры по тексту, payload, событиям (лайк, вступление в группу и т.д.).

### Node.js + vk-io

Альтернатива для JavaScript-разработчиков:

```javascript
const { VK } = require('vk-io');

const vk = new VK({ token: process.env.VK_TOKEN });

vk.updates.on('message_new', async (context) => {
  if (context.text === 'Привет') {
    await context.send('Привет! Напишите /помощь для списка команд');
  }
});

vk.updates.start();
```

### Хостинг и деплой

VK-бот должен работать круглосуточно. Варианты деплоя:
- **Callback API (webhook)** — VK отправляет POST-запросы на ваш сервер при каждом событии. Требует публичного URL (HTTPS)
- **Long Poll** — бот сам опрашивает сервер VK каждые несколько секунд. Проще в разработке, подходит для начала

Я использую VPS (обычно от 500 ₽/мес), настраиваю systemd-сервис или Docker-контейнер для автоматического перезапуска при сбоях.

## Когда выбрать VK-бот, а когда Telegram-бот?

### Выбирайте VK, если:

✅ Ваша аудитория активна во ВКонтакте (особенно регионы, B2C, мода, красота, местный бизнес)
✅ У вас уже есть раскрученное VK-сообщество с подписчиками
✅ Нужно работать с рассылками по существующей базе ВКонтакте
✅ Основные клиенты — пользователи 30+ лет

### Выбирайте Telegram, если:

✅ Аудитория — IT, предприниматели, молодёжь 18–35 лет
✅ Нужен бот в чатах/каналах (Telegram лучше справляется с групповой динамикой)
✅ Требуется сложная логика: оплаты, mini apps, файловый обмен
✅ Ваши клиенты уже есть в Telegram

### Делайте оба, если:

Бизнес работает с широкой аудиторией и хочет максимального охвата. Я разрабатываю ботов на обеих платформах с общим бэкендом — это снижает затраты на поддержку примерно в 1,5 раза.

## Вопросы и ответы

**Нужен ли мне VK-токен и куда его взять?**
Токен выдаётся в настройках вашего VK-сообщества (Управление → Работа с API). Я помогу настроить всё с нуля.

**Бот будет работать 24/7?**
Да. Бот живёт на VPS-сервере и работает круглосуточно. Включена базовая настройка мониторинга.

**Можно ли обновить бота после запуска?**
Да. Я предоставляю поддержку после сдачи и могу добавлять новые функции по договорённости.

**Бот будет работать в нескольких сообществах?**
Стандартно бот привязан к одному сообществу. Поддержка нескольких сообществ — обсуждается отдельно.
