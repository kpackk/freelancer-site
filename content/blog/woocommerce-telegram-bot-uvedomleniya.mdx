Вы запустили интернет-магазин на WordPress + WooCommerce. Заказы идут — но узнаёте о них через email, который проверяете раз в несколько часов. Менеджер не успевает реагировать. Клиенты не понимают, на каком этапе их заказ.

Telegram-интеграция для WooCommerce решает это: новый заказ — мгновенное сообщение вам в Telegram. Клиент изменил адрес — вы уже знаете. Статус сменился на «Отправлен» — клиент получил трекинг автоматически.

В этой статье расскажу: что умеет WooCommerce Telegram-бот, как это работает технически и сколько стоит разработка.

## Зачем нужна интеграция WooCommerce с Telegram

Email-уведомления о заказах — стандарт, который работал 10 лет назад. Сегодня проблемы очевидны:

- Менеджер не открывает почту 2–3 часа → клиент ждёт подтверждения → уходит к конкурентам
- Нет мобильного уведомления → важный заказ замечен поздно
- Email уведомления клиенту попадают в спам
- Нет возможности быстро ответить на вопрос из письма

**Telegram-интеграция даёт:**
- Мгновенные push-уведомления на телефон при новом заказе
- Полную информацию о заказе прямо в мессенджере
- Автоматические уведомления клиенту о статусе заказа
- Возможность управлять заказами из Telegram без входа в админку

Каждое изменение статуса (Обрабатывается → На комплектации → Отправлен → Доставлен) — новое уведомление с деталями.

**3. Отмена заказа**
```

Сумма: 12 840 ₽
```

**4. Новый отзыв**
```
⭐️⭐️⭐️⭐️⭐️ Новый отзыв
Иван Петров о товаре "Кроссовки Nike Air Max":
"Отличные кроссовки, быстрая доставка, соответствуют размеру"
```

**5. Проблемы с оплатой**
Уведомление при неуспешной попытке оплаты — чтобы можно было связаться с клиентом и помочь завершить покупку.

### Уведомления для клиентов

**Клиент получает сообщения в Telegram на каждом этапе:**

| Статус заказа | Что получает клиент |
|---------------|---------------------|
| Заказ принят | Подтверждение, список товаров, сумма |
| Заказ обрабатывается | «Ваш заказ в работе, подготовим к отправке» |
| Отправлен | Номер трекинга, ссылка на отслеживание |
| Доставлен | Просьба оставить отзыв |
| Отменён | Информация об отмене и возврате |

Клиент сам выбирает: хочет ли получать уведомления в Telegram (opt-in при оформлении заказа или через сообщение боту).

### Управление заказами из Telegram

Для небольших магазинов можно добавить возможность управлять заказами прямо из Telegram:

- `/orders` — список новых заказов
- `/order 1547` — детали заказа #1547
- `/status 1547 shipped` — сменить статус на «Отправлен»
- `/note 1547 Позвонить клиенту утром` — добавить заметку к заказу

Это удобно: не нужно открывать ноутбук, чтобы обработать заказ, поступивший вечером.

## Как это работает технически

### Архитектура интеграции

```
WooCommerce (WordPress)
    ↓ WooCommerce Hooks (woocommerce_order_status_changed)
PHP-скрипт / WordPress плагин
    ↓ HTTPS POST
Telegram Bot API
    ↓
Telegram-чат менеджера / группа / клиент
```

Два подхода к реализации:

**Вариант 1 — WordPress плагин (PHP)**
Кастомный плагин перехватывает хуки WooCommerce и напрямую отправляет сообщения через Telegram Bot API. Самый быстрый и надёжный способ, так как работает в рамках WordPress.

**Вариант 2 — Внешний микросервис**
WordPress отправляет webhooks на отдельный Python/Node.js сервис, который обрабатывает события и управляет Telegram-ботом. Этот подход даёт больше гибкости: можно добавить базу данных, аналитику, сложные сценарии.

Для большинства магазинов достаточно Варианта 1 — он проще и не требует дополнительного сервера.

### Пример кода: уведомление о новом заказе (PHP)

```php
// functions.php вашей темы или кастомный плагин

add_action('woocommerce_order_status_processing', 'notify_telegram_new_order');

function notify_telegram_new_order($order_id) {
    $order = wc_get_order($order_id);
    $bot_token = get_option('telegram_bot_token');
    $chat_id = get_option('telegram_admin_chat_id');

    $items = [];
    foreach ($order->get_items() as $item) {
        $items[] = "• " . $item->get_name() . " — " . $item->get_quantity() . " шт";
    }

    $message = "🛒 Новый заказ #{$order_id}\n";
    $message .= "Покупатель: " . $order->get_billing_first_name() . " " . $order->get_billing_last_name() . "\n";
    $message .= "Телефон: " . $order->get_billing_phone() . "\n";
    $message .= "Товары:\n" . implode("\n", $items) . "\n";
    $message .= "Итого: " . $order->get_total() . " ₽\n";
    $message .= "Оплата: " . $order->get_payment_method_title();

    $url = "https://api.telegram.org/bot{$bot_token}/sendMessage";
    wp_remote_post($url, [
        'body' => [
            'chat_id' => $chat_id,
            'text'    => $message,
            'parse_mode' => 'HTML',
        ]
    ]);
}
```

Дополнительно добавляются кнопки (InlineKeyboardMarkup) для быстрых действий прямо из уведомления.

## Часто задаваемые вопросы

**Нужен ли мне отдельный сервер?**
Для базового варианта (плагин WordPress) — нет, всё работает на вашем хостинге. Для расширенного функционала с базой данных и сложной логикой может потребоваться VPS (от 500 ₽/мес).

**Это безопасно? Telegram будет знать о моих заказах?**
Бот — это ваш, созданный через официальный BotFather. В Telegram передаются только данные, которые вы решите передавать. Персональные данные клиентов не нужно указывать в уведомлениях — достаточно номера заказа и суммы.

**Работает ли это с любой темой WordPress?**
Да. Интеграция использует стандартные хуки WooCommerce, которые работают независимо от темы.

**Что если Telegram временно недоступен?**
Добавляем обработку ошибок и retry-логику: если Telegram API не ответил — попробуем снова через 30 секунд. Заказ при этом сохраняется в WooCommerce в обычном режиме.

**Можно ли уведомлять нескольких менеджеров?**
Да. Бот может отправлять уведомления в личные чаты нескольким людям или в общую Telegram-группу менеджеров.
